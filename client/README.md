    - `$ npm run dev` : Port 4200
    - `$ npm run lint`

    - `$ npm run build`
    - Check `./src/environments/environment.ts` file
- Start Server (Client Only)
    - `$ npm start` : Port 8080
- Make sure WSL IP is public
