## Start Server

- Tested On
    - `$ node --version` : `14.20.0`
    - `$ npm --version` : `6.14.17`

```bash
# Client
$ cd ./client/
$ npm install
$ npm run build

# Server
$ cd ./server/
$ npm install
$ npm run build

# Start Server At Port 2323 (Default)
$ cd ./server/
$ nohup npm start &
# or
$ nohup node ./dist/main.js &
```
